Privacy Policy — Search With ChatGPT

Effective date: 2026-04-23

Search With ChatGPT does not collect, store, share, or sell personal information.

What this extension does

This extension adds one context menu option ("Buscar con ChatGPT") when the user selects text on a webpage. When clicked, it opens a new tab to ChatGPT including the selected text as a query parameter.

Data handling

  • No analytics
  • No tracking
  • No remote database
  • No user authentication
  • No cookies set by the extension

Selected text is handled locally by the extension only to build the URL opened in the browser.

Permissions used

  • contextMenus: to create the context menu option when text is selected.

Third-party services

When the user clicks the context menu option, a ChatGPT page is opened (chatgpt.com). Any interaction with that site is governed by OpenAI's own terms and privacy policy.
