# Chrome Web Store Listing Draft

## Name
Search With ChatGPT

## Short description
Search selected text instantly with ChatGPT from the right-click menu.

## Detailed description
Search With ChatGPT adds a simple context menu action to Chrome.

How it works:
1) Select text on any webpage.
2) Right-click and choose "Buscar con ChatGPT".
3) A new tab opens in ChatGPT with your selected text.

This extension is lightweight and focused on one thing: faster access to ChatGPT from selected text.

## Privacy
This extension does not collect or store personal data.

## Permissions justification
contextMenus:
Used to create the right-click context menu option when text is selected.
