NAME
Search With ChatGPT

SHORT DESCRIPTION
Añade “Buscar en ChatGPT” al menú contextual al seleccionar texto en Chrome.

LONG DESCRIPTION
Search With ChatGPT te permite seleccionar texto en cualquier página web, hacer clic derecho y abrir una nueva pestaña con la búsqueda en ChatGPT.

Funciones:
- Añade una opción simple al menú contextual.
- Funciona al seleccionar texto en cualquier web.
- Abre ChatGPT en una pestaña nueva con la consulta ya preparada.
- Sin registros, sin cuenta propia y sin backend del desarrollador.

Ideal para:
- estudiar,
- resumir textos,
- pedir explicaciones,
- traducir contenido,
- investigar rápidamente desde el navegador.

PERMISSIONS JUSTIFICATION
contextMenus:
Se usa solo para mostrar la opción “Buscar en ChatGPT” cuando el usuario selecciona texto.

tabs:
Se usa solo para abrir una nueva pestaña con la URL de ChatGPT y la consulta codificada.

PRIVACY DISCLOSURE SUGGESTION
- Esta extensión no vende datos.
- Esta extensión no transfiere datos a terceros, excepto lo necesario para la función principal al abrir ChatGPT.
- Esta extensión no usa datos para fines ajenos a la funcionalidad descrita.
- Esta extensión no recopila ni almacena datos personales del usuario en servidores del desarrollador.
